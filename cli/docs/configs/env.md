# env.yaml

Defines user project preferences

## Properties

| Property     | Type    | Required | Description                                                                                              |
|--------------|---------|----------|----------------------------------------------------------------------------------------------------------|
| `version`    | integer | **Yes**  |                                                                                                          |
| `fluenceEnv` | string  | No       | Fluence environment to connect to Possible values are: `testnet`, `mainnet`, `stage`, `local`, `custom`. |
