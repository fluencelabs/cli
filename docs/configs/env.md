# env.yaml

Defines user project preferences

## Properties

| Property     | Type   | Required | Description                                                                                           |
|--------------|--------|----------|-------------------------------------------------------------------------------------------------------|
| `version`    | number | **Yes**  |                                                                                                       |
| `fluenceEnv` | string | No       | Fluence environment to connect to Possible values are: `kras`, `testnet`, `stage`, `local`, `custom`. |
